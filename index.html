Twitter Bootstrap Theme

Crystal is the cleanest theme for Twitter Bootstrap which purpose is to deliver the best readability on the whole range of devices


Typography

Crystal Theme is built with a purpose to deliver a better reading experience. It utilizes em-based typography that rely on different number of characters per line (CPL) for different viewport sizes. It is the most important parameter for responsive typography in our fast-changing multi-device world.

Screen Size Characters per Line (Approx.)
Up to 320px 35
321–767px 45
768–990px 60
991–1200px 65
1201px and wider 85

Why so serious?

The typography scale was implemented using the ideas described in the next sources:

  1. “How to Tune Typography Based on Characters Per Line” by Chris Pearson
  2. “A More Modern Scale for Web Typography” by Jason Pamenta

Responsive Web Design is a big deal, and your typography is one of the most noticeable elements in your design.

Customize Crystal

Developing Crystal I didn't touch Twitter Bootstrap files. I just overrided all the variable values as well as extened each module with a custom LESS file. So it makes pretty easy to update Twitter Bootstrap behind and customize the theme itself:

  • Change variable values in less/variables.less (the most important one will be @brand-primary-color, that will affect the main theme color)
  • Rebuild bootstrap-crystal.css compiling it from less/bootstrap-crystal.less
